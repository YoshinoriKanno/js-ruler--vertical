JS Vertical Ruler

Version 1.0

About JS Vertical Ruler

左端に100ピクセル単位のルーラーを表示させます。

jQuery に依存しません。

How to Use

  • vertical-ruler.js を <body> 内に記述

                        <script src="./vertical-ruler.js"></script>

  • vertical-ruler.css を <head> 内に記述

                        <link href="styles/vertical-ruler.css" rel="stylesheet" type="text/css" media="all" />

  • 下記 HTML を <body> 内に記述

                        <div id="ruler" class="ruler"></div>
